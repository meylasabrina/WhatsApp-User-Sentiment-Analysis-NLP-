# Play Store Reviews Sentiment Analysis

A machine learning project that analyzes Google Play Store app reviews and classifies them into positive, negative, or neutral sentiment.

---

## 📌 Project Overview

This project performs sentiment analysis on user reviews from the Google Play Store using Natural Language Processing (NLP) and Machine Learning techniques. The goal is to extract insights from user feedback and understand public opinion toward mobile applications.

---

## 🧠 Features

- Text data cleaning and preprocessing
- Case folding, punctuation removal, and stopword removal
- Feature extraction using TF-IDF Vectorizer
- Sentiment classification (Positive, Negative, Neutral)
- Model training and evaluation
- Data visualization of sentiment distribution
